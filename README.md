# Простой веб-сервер

Веб-сервер на C++ с двумя страницами: авторизация и магазин яблок.

## Структура проекта

```
vulnerable-server/
├── src/
│   ├── server.cpp          # Основной файл сервера
│   └── http_handler.h/cpp  # Обработка HTTP запросов
├── lib/
│   └── shop.h/cpp          # Библиотека магазина (работа с пользователями)
├── templates/
│   ├── login.html          # HTML шаблон страницы авторизации
│   └── main.html           # HTML шаблон главной страницы
├── Makefile                # Файл сборки
└── README.md
```
